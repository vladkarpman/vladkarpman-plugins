# Example Test Templates

These are **example templates** you can copy and customize for your tests.

## Available Templates

| Template | Use Case |
|----------|----------|
| `basic.test.yaml` | Minimal starting point |
| `login.test.yaml` | Authentication flows |
| `navigation.test.yaml` | Screen navigation |
| `form-input.test.yaml` | Form submission |
| `onboarding.test.yaml` | Multi-step onboarding |
| `search.test.yaml` | Search functionality |

## Usage

Copy a template to your tests folder and customize:

```bash
cp templates/login.test.yaml tests/my-login.test.yaml
```

Or just use `/create-test` command which generates a basic template automatically.

## Note

These files are reference examples, not used by any commands directly.
