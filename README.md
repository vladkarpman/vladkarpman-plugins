# Vladkarpman Plugins

Claude Code plugins by Vladislav Karpman.

## Installation

```bash
# Add this marketplace (one time)
claude plugin marketplace add vladkarpman/vladkarpman-plugins

# Install any plugin
claude plugin install <plugin-name>
```

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [mobile-ui-testing](https://github.com/vladkarpman/mobile-ui-testing) | YAML-based mobile UI testing framework with mobile-mcp | 2.0.0 |

## Quick Start

```bash
# Add marketplace
claude plugin marketplace add vladkarpman/vladkarpman-plugins

# Install mobile-ui-testing
claude plugin install mobile-ui-testing

# Restart Claude Code to activate
```
